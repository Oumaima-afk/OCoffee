# O'Coffee

Bienvenue sur le dépôt du projet **O'Coffee**, un site vitrine pour une boutique de café haut de gamme située à Hauts-de-Cloques.

## Description

**O'Coffee** est un détaillant de café et de produits associés à la consommation du café. Ce projet vise à fournir un site web informatif permettant aux clients de découvrir les produits disponibles en boutique, ainsi que des informations sur l'ambiance et le savoir-faire de la boutique.

Le site est conçu pour être accessible, écologique et responsive, afin d'offrir une expérience utilisateur optimale sur tous les appareils.

## Fonctionnalités

- **Page d'accueil** : Présentation de la boutique et des dernières nouveautés.
- **Catalogue** : Liste des cafés disponibles avec leurs caractéristiques principales.
- **Détail produit** : Informations détaillées sur chaque café, y compris son origine, ses caractéristiques et sa disponibilité.
- **Boutique** : Présentation de l'ambiance et du savoir-faire de la boutique.
- **Contact** : Formulaire pour contacter la boutique.

## Technologies utilisées

- Backend : Node.js avec Express
- Base de données : PostgreSQL
- Frontend : EJS pour les templates, HTML/CSS/JavaScript
- Cartographie : Leaflet

## Structure du projet

- /public => Fichiers statiques (CSS, JS, images)
- /views => Templates EJS
- /routes => Routes Express
- /controllers => Logique métier
- /datamappers => Connexion avec la base de donnée
